Manipulación del DOM en JavaScript

El DOM (Document Object Model) nos permite acceder y modificar los elementos HTML mediante JavaScript.

📌 Selección de Elementos

  • Por ID: 
    document.getElementById("miElemento");
  • Por clase: 
    document.getElementsByClassName("miClase");
  • Por etiqueta: 
    document.getElementsByTagName("p");
  • Con querySelector: 
    document.querySelector(".miClase");
