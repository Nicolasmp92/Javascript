Funciones en JavaScript

Las funciones en JavaScript nos permiten reutilizar código. Hay diferentes formas de definir una función:

📌 Tipos de Funciones

  • Función tradicional: 
    function saludar() {
    return "Hola, mundo!";
}
  • Función anónima: 
    const sumar = function(a, b) {
    return a + b;
};
  • Arrow function: 
    const multiplicar = (a, b) => a * b;
